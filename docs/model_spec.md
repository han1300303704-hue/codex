# 仿真模型与评价指标

## 近场信道与运动状态

ULA 阵元位置为 `x_n=(n-(N-1)/2)d`，用户位置为 `p=[r sin(theta), r cos(theta)]^T`。第 `n` 个阵元到用户的距离为：

```text
rho_n = sqrt((x_n-r sin(theta))^2 + (r cos(theta))^2).
```

窄带 LoS 阵列响应采用球面波相位 `exp(-j*2*pi*(rho_n-r)/lambda)`，并保留逐阵元 `1/rho_n` 幅度变化。阵元非均匀多普勒由用户速度向量与各阵元视线方向的投影得到，用作运动感知诊断。

状态为：

```text
x_k = [r_k, theta_k, v_r,k, v_t,k, Delta_f_k, phi_k]^T.
```

其中前四项按极坐标自由运动方程推进；CFO 是慢变随机游走，公共相位满足 `phi_(k+1)=phi_k+2*pi*Delta_f_k*T+nu_k`，且 `nu_k` 为由相位噪声线宽确定的 Wiener 增量。

## 导频观测与 EKF

每个跟踪时隙包含多个已知导频。第 `m` 个导频的阵元观测为：

```text
y_m = sqrt(P) h(r,theta) exp(j*(phi + 2*pi*Delta_f*tau_m)) + w_m.
```

初始导频突发的相邻阵元向量相关性给出 CFO 粗估计；以预测球面波响应投影并去除 CFO 后得到公共相位估计。EKF 使用上述复观测的实部和虚部拼接向量；为了避免对高维阵元观测求逆，更新步采用信息形式，仅反演 6 x 6 状态信息矩阵。

在首次 EKF 更新前，接收端围绕粗先验进行有限范围的距离-角度波束扫描，以导频投影幅度选出初始焦点。这一步避免较大的初始几何误差直接进入非凸相位观测模型而导致局部发散。

## 量化波束、失锁与指标

连续球面聚焦相位被映射至 `2^B` 个等间隔相位级别。联合算法在预测协方差确定的局部距离-角度候选集中，最大化不确定区域的平均增益，同时惩罚守卫区域响应。连续 3 个时隙的归一化增益低于 0.5 时判为失锁，下一时隙进行宽范围码本重捕获。

输出指标包括：归一化波束增益、位置/距离/角度 RMSE、失锁概率、平均失锁事件数和平均重捕获次数。鲁棒性扫描固定使用联合算法，分别改变量化位宽和相位噪声线宽。
